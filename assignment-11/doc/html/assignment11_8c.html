size_t binsearch_missing ( const array *  sparse)

Find the index of the first element after the ``gap''.

Parameters
sparseThe array to look in
Returns
the index of the element right after the first gap.

This task is equivalent to finding the shortest dense prefix of the given array, where by ``dense'' I mean that all elements in this array are in increasing order and there is no three elements N1, N2 and N3 s.t. N1 > N2 > N3 and N2 is not a member of the given array, but N1 and N3 are.
