Detailed Description

This file defines functions and data-structures needed to represent the query language used in library interactions.

Queries are instantiated by the code generated by YACC from the grammar you can find in query_grammar.y

Parameters
kindThe action requested by this query.

patron_idThe unique identifier for the patron on whose behalf this query is perfomred.
book_idThe uniqu identifier of the book operated on.
