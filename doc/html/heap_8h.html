Detailed Description

This file defines basic functions for working with printable heaps.

This file contains declarations for a specific kind of binary trea also known as a heap. Two different implementations are given, the leas_heap uses as list as its underlying structure, while array_heap uses arrays. The list-based implementation is more careful in allocating memory while array-based implementation offers somewhat improved access times.
