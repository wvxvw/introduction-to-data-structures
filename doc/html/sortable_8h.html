This file defines generic functions which are shared by some sortable data structures such as array, list, queue and stack.

typedef size_t rationalization_fn_t(printable *elt, printable *min, printable *max, size_t range)

Functions of this type must translate elt to an integral value in the range between the intergral value of min and the integral value of max and project it onto the range.

Parameters
eltThe element whose value is being computed.
minThe element whose value should be the lowest possible value for elt.
maxThe element whose value should be the highest possible value for elt.
rangeThe returned value should be scaled to fit in the range. In other words, 0 <= f(elt) <= range.

Removes the given element from the container.

For elements which index their elements, the key parameter indicates which element to remove. For containers such as list, the first element is removed to ensure the constant time of this function.
